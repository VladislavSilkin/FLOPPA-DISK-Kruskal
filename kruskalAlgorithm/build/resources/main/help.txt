Для запуска программы, импортируйте граф через File->Open, и нажмите на кнопку Fast End, тогда результат выведется на экран, и его можно будет сохранить. 
Текущий граф распологается в середине интерфейса программы. Чтобы сохранить текущий граф, нажмите File->Save. 
Для загрузки графа из файла используйте File->Open.
Запуск алгоритма по шагам происходит от кнопки Start, после чего можно будет использовать кнопки Next Step и Previous Step для перехода по шагам алгоритма.
